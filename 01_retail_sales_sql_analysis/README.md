# 🛍️ Retail Sales SQL Analysis

This project uses **SQL** to analyze retail sales data and generate insights to support business decisions.

## 🧰 Tools Used
- MySQL Workbench
- Excel (for charts & summaries)
- ChatGPT for query brainstorming

## 🔍 Business Questions Answered
- What are the top-selling products and categories?
- Which regions generate the highest revenue?
- What’s the average order value by customer segment?
- Who are the most valuable customers?

## 📊 Key SQL Skills Demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY, JOINs, Aggregate Functions
- Subqueries, CASE WHEN, RANK

## 📁 Files in this folder
- `retail_dataset.csv` – Sample or mock data
- `retail_analysis.sql` – All queries used
- `retail_summary.xlsx` – Excel visuals
- `Retail_Project_Report.pdf` – Final insights summary
